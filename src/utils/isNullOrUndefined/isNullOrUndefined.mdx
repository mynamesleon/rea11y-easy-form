# `isNullOrUndefined()`

Nothing special here. Just a basic utility function to check if the provided argument is `null` or `undefined`.

## Usage

```js
import { isNullOrUndefined } from 'rea11y-easy-form';

isNullOrUndefined(true); // false
isNullOrUndefined(null); // true
```

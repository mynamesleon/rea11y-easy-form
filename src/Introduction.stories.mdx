`rea11y-easy-form` is a small component library, intended to make setting up
forms in React as easy as possible.

## Install

Assuming you already have `react` and `react-dom` installed:

```shell
npm i rea11y-easy-form react-final-form final-form
# or
yarn add rea11y-easy-form react-final-form final-form
```
